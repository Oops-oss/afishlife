# 测测你的隐藏人格

一个精美的隐藏人格心理测试网页应用，12道题揭示最真实的自己。

## 功能特点

- 🎭 **12道心理测试** - 揭示你隐藏的人格类型
- ⏱️ **快速测试** - 约3分钟完成
- 🌟 **12种人格类型** - 深渊独行侠、微笑摆烂王、暗夜病娇...
- ✨ **精美界面** - 玄学神秘风格，星空粒子动效

## 人格类型

| 类型 | 特点 |
|------|------|
| 深渊独行侠 | 表面高冷，内心戏超多 |
| 微笑摆烂王 | 永远笑嘻嘻，内心OS超多 |
| 暗夜病娇 | 安静时天使，发病时偏执 |
| 人间清醒 | 看透一切，活得明白 |
| 疯批美人 | 美丽且疯狂，极端吸引人 |
| 高敏玻璃心 | 共情能力强，一句话脑补一部剧 |
| 反卷达人 | 佛系大佬，凡事差不多就行 |
| 戏精本精 | 一人分饰多角 |
| 掌控欲怪 | 表面温柔，实则掌控一切 |
| 纯爱战士 | 相信爱情，要求极高 |
| 孤独患者 | 渴望被爱又害怕受伤 |
| 觉醒者 | 经历过低谷，然后重生 |

## 技术栈

- HTML5 + CSS3
- Vanilla JavaScript
- Canvas 星空粒子动画
- 响应式设计

## 快速部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Oops-oss/hidden-personality-test)

## 本地运行

```bash
# 克隆项目
git clone https://github.com/Oops-oss/hidden-personality-test.git

# 进入目录
cd hidden-personality-test

# 使用任意 HTTP 服务器运行
# Python
python -m http.server 8000

# 或者 Node.js
npx http-server -p 8000
```

然后访问 http://localhost:8000
