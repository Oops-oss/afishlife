# 霍兰德职业兴趣测试

基于约翰·霍兰德（John Holland）的职业兴趣理论开发的在线测评工具。

## 功能特性

- 📝 60道精选题目，覆盖RIASEC六个维度
- ⏱️ 约5-8分钟完成测试
- 📊 Canvas绘制动态雷达图
- 🎨 活力渐变视觉设计
- 📱 完全响应式，支持移动端

## RIASEC 六维度

| 代码 | 类型 | 特点 | 典型职业 |
|------|------|------|----------|
| R | 实际型 | 动手能力强、务实 | 工程师、技术工人 |
| I | 研究型 | 分析、理性、好奇 | 科学家、研究员 |
| A | 艺术型 | 创意、表达、独特 | 设计师、艺术家 |
| S | 社会型 | 人际、关怀、助人 | 教师、咨询师 |
| E | 企业型 | 领导、竞争、冒险 | 管理者、创业者 |
| C | 常规型 | 细致、有序、规则 | 会计、行政 |

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需服务器。

## 评分机制

- 喜欢 = 2分
- 一般 = 1分
- 不喜欢 = 0分

每维度10道题，最高20分。结果取得分最高的前三个维度生成三字母代码。

## 技术实现

- 纯原生实现，无外部依赖
- Canvas API 绘制雷达图
- CSS3 动画和过渡效果
- 语义化HTML结构
