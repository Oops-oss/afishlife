# 你的恋爱属性是什么

一个精美的恋爱属性性格测试网页应用，12道情景题揭示你在爱情中的真实模样。

## 功能特点

- 💕 **12道情景测试** - 揭示你的恋爱属性类型
- ⏱️ **快速测试** - 约3分钟完成
- 🌸 **8种恋爱属性** - 甜蜜粘人精、独立女王、浪漫诗人...
- ✨ **精美界面** - 浅粉清新风格，少女心满满

## 恋爱属性类型

| 类型 | 特点 |
|------|------|
| 纯爱战士 💕 | 专一深情，一心一意的理想主义者 |
| 恋爱脑本脑 🧠 | 奋不顾身，为爱燃烧的奉献型 |
| 清醒海王 🌊 | 魅力四射，但心里有杆秤 |
| 智性恋者 📚 | 灵魂共鸣大于一切 |
| 甜蜜猎手 🎯 | 懂得经营的恋爱高手 |
| 佛系恋人 🍃 | 随缘不强求的平和派 |
| 患得患失型 🎭 | 越爱越焦虑的矛盾体 |
| 恋爱绝缘体 ⚡ | 独立自我的单身贵族 |

## 技术栈

- HTML5 + CSS3
- Vanilla JavaScript
- 响应式设计

## 本地运行

```bash
# 进入目录
cd love-attribute-test

# 使用 Python HTTP 服务器运行
python -m http.server 8000
```

然后访问 http://localhost:8000
